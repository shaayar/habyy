# Habby Brand Book 🌱

## 1. Brand Overview

**Brand Name:** Habby  
**Category:** Habit Tracker / Productivity / Personal Growth  
**Platform:** Web App  
**Primary Tone:** Calm, supportive, rewarding  
**Core Theme:** Small daily actions that lead to meaningful growth

Habby is a cozy, modern habit tracking app designed to make consistency feel approachable and emotionally rewarding.

It is not built around pressure, guilt, or “grind culture.”  
Instead, Habby is built around:

- gentle accountability
- visible progress
- sustainable routines
- positive reinforcement

---

## 2. Brand Essence

> **Tiny actions, big becoming.**

This is the emotional core of Habby.

Habby should help users feel like:

- progress is possible
- consistency can be calm
- missing a day is not failure
- habits are something you grow into

---

## 3. Brand Positioning

### Positioning Statement
Habby is a soft, growth-themed habit tracker that helps users build consistency through small daily wins.

### What Habby is
- supportive
- lightweight
- friendly
- modern
- structured but not strict

### What Habby is NOT
- aggressive
- hustle-focused
- guilt-driven
- robotic
- hyper-corporate

---

## 4. Brand Personality

If Habby were a person, it would be:

- encouraging
- tidy
- emotionally intelligent
- patient
- cheerful without being loud
- productive without being preachy

### Personality Traits
- warm
- calm
- playful
- growth-oriented
- non-judgmental
- thoughtful

---

## 5. Brand Archetype

### Primary Archetype: The Companion
Habby supports users with encouragement and emotional ease.

### Secondary Archetype: The Gardener
Habits are treated like things that grow over time through small acts of care.

This gardening metaphor should subtly influence the product language and visuals.

---

## 6. Visual Theme

### Core Creative Direction
> **A garden of routines**

Habits are treated like:

- seeds
- sprouts
- small blooms
- things that need gentle consistency

This theme should appear subtly through:

- iconography
- microcopy
- onboarding
- streak language
- decorative elements

---

## 7. Color Palette

Habby’s palette is designed to feel:

- fresh
- calming
- rewarding
- organic
- clean

It should never feel:

- sterile
- overly bright
- corporate
- chaotic

---

# Primary Colors

## 1) Habby Mint
**Hex:** `#7BDCB5`

### Meaning
Freshness, growth, completion, vitality.

### Use for
- primary buttons
- active states
- progress bars
- selected toggles
- completion states
- links
- highlighted icons

### Avoid
- using it as body text on white
- filling too many large sections with it at full strength

---

## 2) Leaf Green
**Hex:** `#2F8F6B`

### Meaning
Stability, grounding, trust, maturity.

### Use for
- hover states
- stronger CTA variants
- charts
- completed stat cards
- headings on tinted surfaces
- strong success UI

### Avoid
- overusing it for large backgrounds
- mixing it too heavily with multiple saturated colors on the same screen

---

## 3) Soft Cream
**Hex:** `#FFF9F1`

### Meaning
Warmth, comfort, calm background energy.

### Use for
- main app background
- section backgrounds
- warm neutral surfaces
- empty states
- onboarding backgrounds

### Avoid
- using too many cream-on-cream layers without contrast

---

## 4) Cloud White
**Hex:** `#FFFFFF`

### Meaning
Clarity, cleanliness, spaciousness.

### Use for
- cards
- panels
- forms
- modals
- habit tiles
- elevated UI sections

### Avoid
- placing too many pure-white blocks without subtle borders/shadows

---

## 5) Pebble Gray
**Hex:** `#7C857E`

### Meaning
Supportive structure, quiet readability.

### Use for
- secondary text
- labels
- muted icons
- helper copy
- placeholders
- low-emphasis metadata

### Avoid
- using for important CTA text
- using as the only text color in important UI

---

## 6) Forest Ink
**Hex:** `#1F2A24`

### Meaning
Trust, depth, readability.

### Use for
- headings
- body text
- major labels
- important UI text
- dashboard numbers

### Avoid
- using pure black instead unless absolutely necessary

## 8. Recommended Color Usage Rules

To keep the interface cohesive:

### Use Habby Mint for:
- main CTA buttons
- active tabs
- completion toggles
- progress UI
- positive interaction states

### Use Leaf Green for:
- stronger emphasis
- hover/pressed states
- charts
- completed metric cards
- deeper contrast accents

### Use Soft Cream + Cloud White for:
- most surfaces
- layout backgrounds
- cards
- app sections

### Use Forest Ink + Pebble Gray for:
- all text hierarchy
- readable dashboards
- labels and metadata

### Use Sunbeam sparingly for:
- streaks
- rewards
- celebration states
- completion delight

### Use Coral sparingly for:
- warnings
- missed progress
- delete actions
- “not completed” indicators

### Use Lavender / Sky as support colors for:
- category tags
- analytics blocks
- decorative UI accents

---

## 9. Color Pairing Guidance

### Best pairings
- Habby Mint + Soft Cream
- Leaf Green + Cloud White
- Forest Ink + Soft Cream
- Sunbeam + Cloud White
- Lavender Mist + Forest Ink
- Sky Dew + Forest Ink

### Avoid pairings
- Coral + Sunbeam in large sections
- Mint + Sky + Lavender + Coral all on the same card
- using every accent color on the same screen

### General Rule
Each screen should usually have:
- 1 primary brand color
- 1 support accent
- neutrals doing most of the heavy lifting

---

## 10. Dark Mode Guidance (for future)

Dark mode should feel like:

> **a moonlit greenhouse**

Not a hacker dashboard.

### Suggested dark palette

- Background: `#121815`
- Surface: `#1A231E`
- Card: `#223029`
- Primary Accent: `#7BDCB5`
- Main Text: `#F4F8F5`
- Secondary Text: `#AAB7AE`

### Dark mode rules
- keep contrast high
- reduce visual noise
- use mint accents more selectively
- avoid glowing neon saturation

---

## 11. Typography

### Primary Font
**Inter**

### Why
- modern
- clean
- readable
- excellent for apps and dashboards

### Optional Display Font
**Sora**

Use only for:
- hero headings
- marketing sections
- homepage moments

Avoid using display fonts inside the app dashboard UI too heavily.

---

## 12. Typography Style Rules

### Headings
- clean and confident
- slightly soft, not too sharp
- semi-bold or bold

### Body text
- readable and airy
- generous spacing
- calm rhythm

### UI text
- compact but not cramped
- friendly and clean

### Overall typography mood
Modern, breathable, quietly optimistic.

---

## 13. Shape Language

Habby should feel rounded and approachable.

### Use
- large rounded cards
- rounded buttons
- pill badges
- soft containers

### Recommended radius direction
- cards: large rounded corners
- buttons: rounded
- chips/tags: pill shape

Rounded geometry is a core part of the brand feel.

---

## 14. Motion Style

Animation should feel:

- light
- rewarding
- smooth
- subtle

### Good motion ideas
- completion toggle pop
- progress fill animation
- card fade-in
- streak badge pulse
- button hover lift

### Avoid
- dramatic transitions
- noisy confetti everywhere
- excessive floating animations

Habby should move like a quiet breeze, not a casino.

---

## 15. Iconography

Icon style should be:

- rounded
- simple
- modern
- slightly playful

### Good icon themes
- sprout
- checkmark
- flame
- calendar
- spark
- droplet
- moon
- star
- circle progress

Icons should support the gardening / growth metaphor subtly.

---

## 16. Logo Direction

### Recommended logo concept
A **checkmark + sprout hybrid**

This is the strongest symbol for Habby because it represents:

- completion
- growth
- consistency

### Strong logo options
- checkmark shaped like a leaf
- lowercase “h” with a leaf tip
- sprout inside a rounded circle

### Logo mood
- minimal
- rounded
- modern
- friendly

Avoid logos that feel:
- too childish
- too corporate
- too detailed for small-screen use

---

Habby should sound:

- supportive
- lightly playful

- never judgmental

### Voice qualities
- warm
- human

### Habby should NOT sound

## 19. UI Application Guidance

### Dashboard
Should feel:
- welcoming
- rewarding
- uncluttered

Use:
- progress cards
- today summary
- streak highlights
- completion visuals

### Habit Cards
Should feel:
- lightweight
- tactile
- easy to scan

Use:
- white or soft cream cards
- mint/green completion states
- small accent chips

### Analytics
Should feel:
- insightful
- motivating
- visual but not noisy

Use:
- mint, green, sky, and sunbeam strategically
- clean charts
- strong spacing

### Settings / Utility pages
Should feel:
- simple
- calm
- low-friction

Keep these mostly neutral with minimal accents.

Habby is a **cozy, growth-oriented habit tracker** that helps users stay consistent through small daily actions.

It should feel like:

- a tiny growth companion
- a calm productivity ritual
- a friendly tool for becoming more consistent

### One-line summary
> **A soft, modern habit tracker built around tiny daily wins.**
